# 判决书常见段落模式与关键词索引

## 一、审级识别

### 案号模式
| 审级 | 案号模式 | 示例 |
|------|----------|------|
| 一审 | （年份）XX法民初X号 | （2024）渝01民初54号 |
| 二审 | （年份）XX民终X号 | （2026）渝民终35号 |
| 再审 | （年份）XX民再X号 | （2025）最高法民再XX号 |

### 法院层级
- "XX基层/区人民法院" → 一审法院
- "XX中级人民法院" → 一审或二审法院（需结合案号判断）
- "XX高级人民法院" → 二审或再审法院
- "最高人民法院" → 再审法院

### 判决书结尾识别
- "本判决为终审判决" → 二审/再审判决
- "如不服本判决，可在判决书送达之日起十五日内，向本院递交上诉状" → 一审判决

## 二、段落起始关键词

### 当事人信息
- "原告：" / "被告：" / "第三人："
- "上诉人（一审原告）：" / "被上诉人（一审被告）："
- "法定代表人：" / "委托诉讼代理人："
- "统称" / "以下简称" → 定义当事人简称

### 上诉/答辩部分（二审）
- "上诉请求：" / "上诉人XX上诉请求"
- "事实与理由：" / "理由如下："
- "辩称：" / "答辩称："
- "A公司辩称" / "B公司辩称"

### 查明事实部分
- "一审法院查明事实如下：" → 一审事实
- "二审查明如下事实：" / "二审查明如下：" → 二审补充事实
- "本院另查明：" → 二审补充查明
- "经审理查明" → 通用事实查明
- 小标题模式："一、" "二、" "三、"（阿拉伯数字）→ 事实分类

### 法院认为部分
- "一审法院认为" / "一审法院认定" → 一审裁判理由
- "本院认为" → 当前审级（二审）的裁判理由
- "争议焦点" / "争议焦点问题为：" / "本案的争议焦点"
- 小标题模式 "（一）" "（二）" "（三）" → 争议焦点分类
- "1.是否构成直接侵权" → 子争议焦点
- "2.是否构成帮助侵权" → 子争议焦点

### 判决结果部分
- "一审法院依照……判决如下：" → 一审判决主文
- "本院依照……判决如下：" → 当前审级判决主文
- "一、" "二、" "三、" → 判决主文条目
- "撤销XX人民法院XX号民事判决" → 撤销一审判决
- "改判" → 变更一审判决

### 诉讼费用
- "一审案件受理费" / "二审案件受理费"
- "由XX负担" / "由XX公司负担XX元"

## 三、知产类案件常见争议焦点

### 权属相关
- "是否系本案适格原告" / "是否享有涉案作品著作权"
- "权属链条" / "授权链条"

### 直接侵权相关
- "是否构成直接侵权" / "是否构成信息网络传播行为"
- "是否向公众提供作品" / "是否将作品置于信息网络中"
- "服务器标准" / "用户感知标准" / "实质替代标准"

### 帮助侵权相关
- "是否构成帮助侵权"
- "明知或应知" / "知道或应当知道"
- "是否采取了必要措施" / "是否及时采取必要措施"
- "通知-删除" / "通知-必要措施"
- "注意义务" / "审查义务"

### 不正当竞争相关
- "是否构成不正当竞争"
- "是否违反诚实信用原则" / "是否违反商业道德"

### 损害赔偿相关
- "惩罚性赔偿" / "故意侵权" / "情节严重"
- "许可使用费" / "实际损失" / "侵权获利" / "法定赔偿"
- "合理开支" / "律师费" / "公证费"

## 四、常见法律条文

### 《著作权法》（2020修正）
- 第3条：作品定义
- 第10条：著作权内容（信息网络传播权为第12项）
- 第11条：著作权归属
- 第12条：演绎作品
- 第24条：合理使用
- 第53条：侵权行为
- 第54条：损害赔偿计算（含惩罚性赔偿）

### 《民法典》
- 第1194条：网络侵权一般条款
- 第1195条：通知-删除规则
- 第1197条：明知/应知的帮助侵权

### 最高人民法院司法解释
- 《关于审理侵害信息网络传播权民事纠纷案件适用法律若干问题的规定》
  - 第3条：网络服务提供者分类
  - 第7条：教唆/帮助侵权
  - 第8-14条：明知/应知认定要素
- 《关于审理侵害知识产权民事案件适用惩罚性赔偿的解释》
  - 第1条：适用条件
  - 第4条：情节严重的认定
- 《关于适用〈反不正当竞争法〉若干问题的解释》
  - 第24条：竞合处理规则

## 五、赔偿金额计算常见公式

| 计算方式 | 公式 | 说明 |
|----------|------|------|
| 许可费参照 | 实际许可费 × 侵权时间/许可时间 | 参照同类作品许可费 |
| 和解金额参照 | 单案和解金额 × 时间系数 | 参考当事人既往和解 |
| 法定赔偿 | 法定范围内酌定 | 一般上限500万 |
| 惩罚性赔偿 | 基数 × 倍数（1-5倍） | 须满足故意+情节严重 |
